  • Robotics and Autonomous Systems Laboratory
    Vanderbilt University, Nashville, TN
    Research intern
    Summer 2011-2013

    Overview

    As a summer intern at the Robotics and Autonomous Systems Laboratory (RASL), I worked on many projects related to how autonomous systems can be used to improve Autism intervention. According to literature, children with Autism tend to interact better with autonomous systems rather than humans. This is likely because these systems do not exhibit as much emotion and spontaneity as humans do. We used this fact to develop systems that could aid in current Autism intervention protocols.
    This is also where I was first introduced to the world of programming as well. My experience at this research facility played a crucial role in my choice to pursue programming as a profession.

    Projects

    NAO Kinect Interface:  I worked on this project with a few other graduate and undergraduate students over the course of 2 summers. The goal of this project was to interface the NAO Humanoid robot with the Microsoft Kinect motion sensor. We worked on writing algorithms that allowed the robot to respond to human gesture. THe Kinect would detect the human's movements and the data would be analyzed in real time. Then the computer would tell the robot the appropriate course of action. This system was actually developed to help children with Autism. According to literature, these children tend to be less overwhelmed by robots rather than humans due to the robots' limited abilities and emotions. I was able to observe firsthand, how this system aided children with Autism through imitation learning. The robot would perform a gesture and then ask the child to repeat the gesture. If the child was correct, the robot would congratulate him and move to the next gesture. Otherwise, the robot would show what the child did wrong and would let the child try again. This project was presented at a conference:
    “Gesture Recognition using Commodity RGB-D Sensor for Imitation Learning Platform for Children with Autism.” HCI International (2013).

    Heart Rate Sensitive Virtual Reality System:  This was the first programming project I worked on at RASL and was one of the biggest reasons why I became interested in computer science. I built a heart rate sensor based off of an online circuit diagram. I then used an Arduino to read the analog output of the sensor with a computer. I filtered the data using Matlab in order to calculate the heart rate. Then I used a pong style game to challenege the user and increase his anxiety. If the heart rate increased different user sensitive thresholds, I had a virtual avatar give an appropriate verbal response. This project was more of a proof of concept about how autonomous systems can be changed based on the user's physiological state. I completed this project over the course of a summer internship and the work was published in a scholarly journal:
    "Development of an Intelligent, Real-time, Heart Rate Sensitive Virtual Reality System." Young Scientist (2012).

  • LeapSynth
    Play music in the air
    Developed at: MHacks Winter 2013
    Developed date: February 2013

    Overview

    The LeapSynth was a project I worked on with 4 other teammates at MHacks Winter 2013, my first ever hackathon. We created a program that used the Leap Motion hand tracking device to play music. The system allowed the user to choose from 128 different instruments, and add effects like reverb to the sound by moving their hands in different directions. We also allowed the user to record whatever they played to a MIDI file so that it could be used in all of the common audio mixing software.
